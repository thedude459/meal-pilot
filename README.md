# Meal Pilot

Family meal planning & grocery automation — Speckit-driven.

## Current feature: Family Member Profiles

Household roster and preference profiles (likes, dislikes, dietary restrictions).

### Quick start

```bash
npm install
npm run db:migrate
npm run dev
```

API: `http://localhost:3000`
